# Quickstart Python Project

If you're in a hurry, or you've been through this before,
this quickstart will get you up and running with a modern
Python project in no time.

## Install Copier

<div class="termy">

```console
$ pip install copier
---> 100%
Successfully installed copier
```

</div>
